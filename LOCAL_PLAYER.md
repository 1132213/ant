# Ant-Game 本地回放播放器 (Local Player) 使用说明

`tools/local_player.py` 是一个基于 `pygame` 的轻量级图形化回放工具，用于读取并可视化 Ant-Game 的对局录像 (`replay.json`)。它能够帮助你直观地分析 AI 的决策、局势的走向以及战术的得失。

## 1. 如何运行

在项目根目录下，使用 Python 运行该脚本：

```bash
# 播放默认路径 (local_match_output/replay.json) 的回放
python tools/local_player.py

# 播放指定路径的回放
python tools/local_player.py path/to/your/replay.json
```

**依赖要求**：需要安装 `pygame` 库 (`pip install pygame`)。

---

## 2. 界面与信息解读

回放器的界面主要分为四个部分：顶部状态栏、主战场地图、全局形势估计曲线、以及底部进度条。

### 2.1 顶部状态栏 (Top UI)

在界面左上角，你可以看到当前回合数以及双方的实时核心数据。

*   **Round**: 当前回合数 / 总回合数。
*   **Coins**: 当前剩余金币。
*   **HP**: 基地剩余血量。
*   **ALv (Ant HP Level)**: 蚂蚁血量科技等级。
*   **SLv (Speed Level)**: 基地出兵速度科技等级。
*   **CD**: 超级武器冷却倒计时，以数组 `[闪电, EMP, 护盾, 回避]` 的形式显示。0 表示就绪。

### 2.2 主战场地图 (Battlefield)

地图使用六边形网格渲染，双方基地分别位于地图的两侧（红方 P0 在左，蓝方 P1 在右）。

*   **地形颜色**：
    *   灰色：普通道路 (`PATH`)
    *   深灰色：障碍物 (`BARRIER`)
    *   浅红/浅蓝色：双方的高地 (`HIGHLAND`)
*   **防御塔**：
    *   显示为带有边框的六边形。
    *   上方文字表示**塔的类型**（如 `Bsc` 代表 Basic, `Mrt` 代表 Mortar）。
    *   下方文字表示**塔的当前血量**。
*   **蚂蚁**：
    *   **圆形**：普通兵种 (Worker 等)。
    *   **方形**：战斗兵种 (Fighter 等)。
    *   **特殊状态边框**：
        *   金色边框：免控状态 (Control Free)
        *   紫色边框：被蛊惑状态 (Bewitched)
        *   青色边框：被冻结状态 (Frozen)
*   **超级武器特效**：
    *   巨大的彩色六边形边框表示正在生效的超级武器区域：黄色(闪电)、蓝色(EMP)、绿色(护盾)、紫色(回避)。

### 2.3 全局形势估计曲线 (Evaluation Curve)

在界面的**右上角**，有一个半透明的黑色浮窗，展示了整场比赛的“胜率走势图”。

*   **黄线走势**：代表 P0 (红方) 的优势。黄线在 0 刻度线上方代表红方优势，在下方代表蓝方优势。
*   **垂直白线**：当前播放的回合游标，随着播放进度向右移动。
*   **左上角数值**：`P0 Eval: xxx.x` 显示了当前具体的回合估值。
*   **估值原理**：该曲线使用了 SDK 默认的 `FeatureExtractor.evaluate` 函数计算得出，综合考量了：**基地血量差 (权重极高)**、科技等级、防御塔阵型覆盖、经济差以及兵线压制情况。

### 2.4 底部进度条 (Progress Bar)

在界面最下方有一个可拖动的进度条。

*   你可以直观地看到当前的播放进度。
*   **点击或拖动**进度条上的滑块，可以立刻将比赛画面跳转到任意指定的回合，方便你反复观看关键的交战瞬间。

---

## 3. 快捷键操作

除了使用鼠标拖动进度条外，你还可以使用键盘进行精确控制：

*   `[Space] (空格键)`: **播放 / 暂停**。
*   `[Right] (右方向键)`: **下一回合**（单步步进）。
*   `[Left] (左方向键)`: **上一回合**（单步回退）。
*   `[Up] (上方向键)`: **增加播放速度**（最高 60 FPS）。
*   `[Down] (下方向键)`: **降低播放速度**（最低 1 FPS）。
